# Siglark Protocol

> 🌐 **Public.** The open protocol for verifiable identity and trust on the physical layer. A Vorim AI Labs product.

Siglark gives every robot and physical AI agent a cryptographic identity, attenuated permission delegation, and an offline counterparty-verifiable record of every action. This repo holds the open specification and examples.

## Why an open protocol

Physical agents take consequential, irreversible real-world actions, and a growing body of regulation requires those actions to be **recorded and tamper-protected** (EU Machinery Regulation 2023/1230, Delegated Reg 2024/2220, UN R155/R156). Verifiability that depends on trusting a single vendor isn't verifiability. The protocol is open so any counterparty — regulator, insurer, operator, or peer robot — can verify a signed record offline, without calling home.
